# 3D to Pixel Art Converter

Application web de conversion de modèles 3D en spritesheets pixel art pour RPG/JRPG.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```

## Docker

```bash
docker-compose up -d
```
